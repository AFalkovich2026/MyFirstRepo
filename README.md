📊 Sales Analysis Dashboard
📌 Project Overview
This project analyzes sales, customers, products, suppliers, and returns using Excel.
The goal is to evaluate business performance and build an interactive dashboard.
📂 Dataset
The Excel file includes the following sheets:
•	Sales_Files
•	Customers
•	Products
•	Suppliers
•	Returns
📊 Key KPIs
•	Total Revenue
•	Target Revenue
•	Revenue Variance
•	Top 5 Products by Revenue
•	Revenue by Category
•	Return Rate
📈 Dashboard Features
•	Pivot Tables & Charts
•	Monthly Revenue Trend
•	KPI Summary Cards
•	Slicers and Timeline for filtering
🛠 Tools Used
•	Microsoft Excel
•	Pivot Tables
•	Pivot Charts
•	Slicers
